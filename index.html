What is the Nostalgia Shader?

This shaderpack is intended to reproduce the look and feel of the "old" shaders from back when the shadersmod was new. However it also builds ontop of that with new features and superior performance. Accidentally it also has some similarities to the now canceled "Super Duper Graphics Pack" and even features a color grading profile for this since version 1.1.
For more information regarding requirements and features you can check the Wiki.
